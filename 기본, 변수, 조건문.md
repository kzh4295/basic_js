# 변수

변수 선언 키워드 : let 
- 초기값을 넣을 수도 있고, 
변수만 선언할 수도 있음

  let 변수명;
  변수명 = 변수값;

-한번 선언한 변수에 다른 변수값을 넣을 수도 있음(재할당 가능)
  let i = 1;
  i = 2;

- 한번 선언한 변수를 다시 선언할 수는 없음
  let j = 1;
  let j = 2;


# 상수

- 상수 선언 키워드 : const

- 상수라고 정수만 가능한 것은 아님

# switch/case 구문도 가능
- case 문에서 break 해주지 않으면 다음 case 문 및 default 문도 실행하게 됨
- 여러개의 변수값을 가지고 각각 실행할 코드가 다른 경우 사용
- break : 해당 case문만 실행해주고 switch 문에서 나옴
- default : case문이 다 안맞을때 사용할 코드 (= else 구문) 
switch(변수값){
  case 1:
    console.log(출력1);
    break;
  case 2:
    console.log(출력2);
    break;
  default:
    console.log(출력3);
}
